class mr.electrical.odeCircuit.OdeComponent(OdeCircuit circuit, list props=[])¶

Bases: object

Installs the component in the circuit, and allocates values for each kwarg, then sets the value.

props - Ordered list of tuples (name, initialValue, maxStableChange).
maxStableChange is the change in value to trigger instability. By default, this value is multiplied by 0.1. For nodes, for instances, maxStableChange == 1.0 for the voltage.
