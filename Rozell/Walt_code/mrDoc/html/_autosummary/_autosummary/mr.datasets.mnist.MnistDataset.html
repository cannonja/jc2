mr.datasets.mnist.MnistDataset¶

class mr.datasets.mnist.MnistDataset(self, path, nLoad=10000000000.0)¶

Bases: mr.datasets.common.ImageSet

Dataset for the MNIST handwritten digit database.

path - Path to train-images-idx3-ubyte and train-labels-idx1-ubyte as
well as t10k-images-idx3-ubyte and t10k-labels-idx1-ubyte. Note that these files must be extracted (not in gzipped form), and for consistency, any dots in the file names must have been changed to dashes.

nLoad - Number of images to load, total.
